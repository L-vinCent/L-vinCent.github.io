启动白屏问题

首先要知道原生 iOS 启动页，它的消失时间就是 didFinishLaunchingWithOptions的函数周期走完，我们平时需要将启动页延迟消失可以采用阻塞主线程来达到目的

知道这个好办，我们可以在 didFinishLaunchingWithOptions里每0.1s休眠一次，直到 RN 中页面加载完成，在 js 文件的 componentDidMount内再调用原生方法取消主线程的休眠方法，这样就达到了目的，具体实现在 RNSplashScreen类
